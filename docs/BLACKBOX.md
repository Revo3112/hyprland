# Hyprland Configuration - Project Context

> **Project Type:** Linux Desktop Environment Configuration  
> **Window Manager:** Hyprland (Wayland compositor)  
> **Base:** end-4/dots-hyprland dotfiles  
> **Shell:** Quickshell (ii config)  
> **Last Updated:** 2026-03-03

---

## Overview

This is a comprehensive Hyprland configuration for an AMD Ryzen APU laptop running Arch Linux. It features a modular configuration structure with custom gaming optimizations, dynamic display management, and integration with Quickshell for a modern desktop experience.

### Key Features

- **Modular Config Structure:** Separated upstream and custom configurations
- **Gaming Optimizations:** Ryzen APU TDP control, GameMode integration, FSR support
- **Dynamic Display Management:** Auto-switching monitor profiles via HyprDynamicMonitors
- **Dual Lock System:** Configurable Hyprlock or Quickshell lock
- **Quickshell Integration:** Modern Qt-based shell with custom toggles

## Configuration Loading Order

Files are loaded in this sequence (later files override earlier):

1. **Upstream defaults:** `hyprland/*.conf`
2. **Custom overrides:** `custom/*.conf`
3. **Display tools:** `monitors.conf`, `workspaces.conf`

### Safe to Modify

| File | Purpose |
|------|---------|
| `custom/env.conf` | Custom environment variables |
| `custom/execs.conf` | Custom startup applications |
| `custom/general.conf` | Monitor & general settings |
| `custom/keybinds.conf` | Custom keybindings |
| `custom/rules.conf` | Custom window rules |
| `monitors.conf` | nwg-displays output |
| `workspaces.conf` | nwg-displays output |

### Do NOT Modify

| File | Risk |
|------|------|
| `hyprland/*.conf` | Will be overwritten on upstream updates |
| `hyprland/colors.conf` | Auto-generated by matugen theme |

## Key Components

### 1. Display Management

**Tools:**
- **HyprDynamicMonitors:** Auto-switch profiles on HDMI plug/unplug
- **nwg-displays:** GUI for monitor configuration

**Profiles:**
- `laptop-only` - Internal display only
- `hdmi-mirror` - Mirror to HDMI
- `hdmi-extend-right` - Extend desktop to HDMI
- `hdmi-only` - External monitor only

**Keybinds:**
```
Super+Alt+F1  → Laptop only
Super+Alt+F2  → Extend right
Super+Alt+F3  → Mirror
Super+Alt+F4  → HDMI only
```

### 2. Gaming Setup

**Hardware:** AMD Ryzen 5 6600H with Radeon 660M (RDNA2)

**TDP Profiles (via ryzenadj-profile):**
| Profile | STAPM | Fast | Use Case |
|---------|-------|------|----------|
| power-saver | 15W | 18W | Battery life |
| balanced | 28W | 35W | Normal use |
| performance | 45W | 54W | Gaming |

**Quickshell Toggles:**
- **Game Mode Toggle:** Enables performance TDP + GameMode + visual optimizations
- **Power Profile Toggle:** Cycles through power profiles

**Environment Variables:**
```bash
AMD_VULKAN_ICD=RADV
RADV_PERFTEST=gpl
VKD3D_CONFIG=dxr
DXVK_ASYNC=1
WINE_FULLSCREEN_FSR=1
```

### 3. Lock System

**Dynamic Lock:** Reads `~/.config/illogical-impulse/config.json` to choose:
- **Hyprlock:** Traditional lock screen
- **Quickshell Lock:** Modern Qt-based lock

**Idle Behavior:**
- 5 min → Lock screen
- 10 min → DPMS off
- 15 min → Suspend

### 4. Keybindings

**Shell Controls:**
```
Super          → Toggle search/overview
Super+Tab      → Toggle overview
Super+V        → Clipboard history
Super+Period   → Emoji picker
Super+A        → Left sidebar
Super+N        → Right sidebar
Super+Slash    → Cheatsheet
Super+K        → On-screen keyboard
Super+M        → Media controls
Super+G        → Overlay
Ctrl+Alt+Del   → Session menu
```

**Monitor Profiles:**
```
Super+Alt+F1-4 → Monitor profile switching
```

## Troubleshooting

### Lock Screen Issues
```bash
# Reset gamemode state
rm -f /tmp/.qs_gamemode_active
rm -f /tmp/.qs_gamemode_inhibit_pid
hyprctl reload
```

### HDMI Not Detected
```bash
# Check port status
cat /sys/class/drm/card1-HDMI-A-1/status

# Reload Hyprland
hyprctl reload
```

### TDP Not Applying
```bash
# Verify ryzenadj installed
which ryzenadj

# Check sudo access
sudo ryzenadj-profile status
```
